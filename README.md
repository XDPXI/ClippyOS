# ClippyOS

A 32-bit operating system based around the windows assistant Clippy

---

## How to run

### Using a pre-compiled ISO

1. Visit the [releases](https://github.com/XDPXI/ClippyOS/releases)
2. Download the ISO from a release

### Building from source

> Required Dependencies:
> - make
> - qemu
> - nasm
> - xorriso

1. Checkout the git repository
```
git clone https://github.com/xdpxi/clippyos
```

2. Compile the OS
```
make
```

3. Run the OS
```
make run
```
